Algunos Tags Explicados

¿ Qué son los Meta tags ?

Las etiquetas"meta" suministran información encriptada a los navegadores y motores de búsqueda sobre un página web, facilitándoles el análisis de los archivos HTML. Con su información guian a los bots ( Crawlers) de los motores de Búsqueda. Son invisibles para el usuario.




Cuales son los Meta-Tags más usados.

a) Meta charset.-

Este Tag se usa para que las "ñ" y las tildes (al igual que otros caracteres especiales) se lean correctamente. El meta tag a usar será: meta charset="utf-8"

a) Meta name description.-

Sirve para describir brevemente el contenido de una página web. Esta información va directo a los motores de búsqueda de internet, convirtiéndose en uno de los más importantes. El meta tag a usar será: meta name="description" content="Aqui descripción".

b) Meta name Author.-

Permiten hacer referencia al diseñador de una página. El meta tag a usar será : meta author="nombre de autor"

c) Meta keywords.-

Son palabras claves en el texto de la web que se usan para que ésta sea fácilmente encontrada. El meta tag a usar será: meta name=”keywords” content= “palabras que deriven de dicha página”.

d) Meta name robots.-

Principalmente para indicarle al bot del motor de búqueda, si es que indexa la página o no. El meta tag será: meta name="robots" content="index" (noindex,follow).

e) Meta name generator.-

Para definir el programa con el que se ha creado el documento. El meta a usar será: meta name="generator" content="Aquí-el-programa".

e) Meta http-equiv.-

Para el control de Caché. Para refrescar una página. Para el expirar.
